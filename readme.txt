== Description ==

Display BuddyPress last active date for a user on `wp-admin/users.php` page

BuddyPress is required for plugin to work. If BuddyPress is not installed - plugin will not display any information.

== Installation ==

1. Upload plugin folder `/buddypress-last-active-users-wp-admin/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to "Users" page in your WordPress admin area and check the new `Last Active` column
